Reefscan Software

The following software is available for use with the reefscan system

Reefscan Android Controller

This app is used to control the reefscan system while it is in operation. It should be installed on a device which can get wet.

It is available for android.

We have a simulated version of the app which demonstrates the features of the app without connecting to a reefscan. Please note that you cannot have the demo and the real app installed on the same device at the same time. Download the demo app here.

Reefscan Data Manager

This app is used to edit metadata and upload the data from the reefscan to hard disks. It should be used indoors.

It is available for windows.
